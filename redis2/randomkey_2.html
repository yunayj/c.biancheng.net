Redis RANDOMKEY命令

 
Redis RANDOMKEY 命令用于从当前数据库中随机返回一个 key。

可用版本

RANDOMKEY 命令可用版本：>= 1.0.0

语法格式

Redis RANDOMKEY 命令的基本语法如下： 
127.0.0.1:6379> RANDOMKEY

返回值

当数据库不为空时，返回一个 key；当数据库为空时返回 nil。

命令演示

127.0.0.1:6379> RANDOMKEY
"webname"
127.0.0.1:6379> SELECT 3
OK
127.0.0.1:6379[3]> RANDOMKEY
(nil)
