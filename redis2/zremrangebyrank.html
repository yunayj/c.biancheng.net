Redis ZREMRANGEBYRANK命令

 
Redis ZREMRANGEBYRANK 命令移除有序集合 key 中，指定排名(rank)区间内的所有成员。其中区间分别以下标参数 start 和 stop 指出，包含 start 和 stop 在内。

下标参数 start 和 stop 都以 0 为底，也就是说，以 0 表示有序集第一个成员，以 1 表示有序集第二个成员，以此类推。您也可以使用负数下标，以 -1 表示最后一个成员， -2 表示倒数第二个成员，以此类推。

可用版本

ZREMRANGEBYRANK 命令可用版本：>= 2.0.0

语法

Redis ZREMRANGEBYRANK 命令的基本语法如下： 
ZREMRANGEBYRANK key start stop

返回值

被移除成员的数量。

命令演示

127.0.0.1:6379> ZADD salary 3000 jack 3500 helen 2880 john 4000 simith 6000 ros
(integer) 5
127.0.0.1:6379> ZREMRANGEBYRANK salary 0 2 #移除前三个排名，默认从小到大排名
(integer) 3
127.0.0.1:6379> ZRANGE salary 0 2 withscores #返回剩余的成员与分数
1) "simith"
2) "4000"
3) "ros"
4) "6000"
