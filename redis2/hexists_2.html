Redis HEXISTS命令

 
Redis HEXISTS 命令用于查看哈希表 key 中，给定域 field 是否存在。

可用版本

HEXISTS 命令可用版本：>= 2.0.0。

语法

Redis HEXISTS 命令的基本语法如下： 
HEXISTS key field

返回值

如果哈希表含有给字段，则返回 1；如果哈希表不含有给定字段，或者是 key 不存在，那么返回 0。

命令演示

127.0.0.1:6379> HSET website name www.biancheng.net
(integer) 1
127.0.0.1:6379> HEXISTS website name  #存在返回1
(integer) 1
127.0.0.1:6379> HEXISTS website age   #不存在返回0
(integer) 0
127.0.0.1:6379> HEXISTS web name
(integer) 0
