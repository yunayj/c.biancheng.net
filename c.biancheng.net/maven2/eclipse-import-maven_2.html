Eclipse导入Maven项目

 
我们知道，m2eclipse 是一个在 Eclipse 中集成 Maven 的插件，通过该插件我们可以很轻松的在 Eclipse 中导入 Maven 项目，本节我们将讲解如何使用 Eclipse 导入 Maven 项目。

导入 Maven 项目

相比新建 Maven 项目，实际工作中使用更多的是将现有的 Maven 项目导入 Eclipse，具体步骤如下。

1）点击菜单栏中的 Import ，开始导入项目，在弹出的对话框中，将 Maven 选项展开，选择 Existing Maven Projects，最后点击 Next 按钮，如图 6 所示。

图6：Eclipse 导入 Maven 项目

2）在弹出的对话框中，选择本地 Maven 项目所在的目录，Ecilpse 会自动识别出该目录下所有 Maven 项目的 pom.xml 文件，我们可以根据情况选择导入一个或多个项目，然后点击 Finish 按钮，如图 7 所示。

图7：选择导入的 Maven 项目

3）返回 Eclipse 工作区，可以看到 Maven 项目已经导入成功，如图 8 所示。

图8：已导入 Maven 项目的目录结构
