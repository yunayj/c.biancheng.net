Java List.get()方法：获取列表指定位置的元素

Java 集合类中的 List.get() 方法用于获取列表指定位置的元素。返回通过索引获取的元素。

语法：

get(int index)

参数说明：
  • index：要返回元素的索引。

典型应用

List 接口的 get 方法在开发中使用非常广泛，本示例使用 List 接口的实现类 ArrayList 初始化一个列表对象 list，并向该列表中添加 3 个元素，然后调用 get 方法获取索引位置为 1 的元素对象。代码如下： 
public static void main(String[] args){
    List<String>list = new ArrayList<String>();
    list.add("保护环境");  //向列表中添加数据
    list.add("爱护地球");  //向列表中添加数据
    list.add("从我做起");  //向列表中添加数据
    String ret = list.get(1);
    System.out.println("获取索引位置为1的元素:"+ret);
}
运行结果如下：
获取索引位置为1的元素：爱护地球
