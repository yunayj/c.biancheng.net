使用C语言打印九九乘法表

我们通常可以使用嵌套循环来打印九九乘法表，比如 C语言中的 for 循环就非常适合这种场景，因为它可以方便地控制迭代次数。

下面是一个使用 C语言打印九九乘法表的例子：
#include <stdio.h>

int main() {
    int i, j;
    for (i = 1; i <= 9; i++) {
        for (j = 1; j <= i; j++) {
            printf("%d*%d=%-2d ", j, i, i * j);
        }
        printf("\n");
    }
    return 0;
}
在这个程序中，我们使用了两个嵌套的 for 循环。外部循环 i 控制行数，从 1 到 9 循环。内部循环 j 控制列数，从 1 到 i 循环。

在循环的每次迭代中，我们使用 printf 函数打印出乘法表中每个元素的值。%d 是用来打印整数的格式化占位符，%-2d 指定要输出的数字长度为 2，左对齐。这样可以保证输出的结果整齐。

当程序运行时，会输出如下结果：

1*1=1 
1*2=2  2*2=4 
1*3=3  2*3=6  3*3=9 
1*4=4  2*4=8  3*4=12 4*4=16
1*5=5  2*5=10 3*5=15 4*5=20 5*5=25
1*6=6  2*6=12 3*6=18 4*6=24 5*6=30 6*6=36
1*7=7  2*7=14 3*7=21 4*7=28 5*7=35 6*7=42 7*7=49
1*8=8  2*8=16 3*8=24 4*8=32 5*8=40 6*8=48 7*8=56 8*8=64
1*9=9  2*9=18 3*9=27 4*9=36 5*9=45 6*9=54 7*9=63 8*9=72 9*9=81

从结果可以看出，我们成功地打印出了一个完整的九九乘法表。

值得注意的是，在打印九九乘法表时，我们还可以调整输出的格式。例如，我们可以在每个数字之间添加一些空格，或者使用制表符来对齐。这样可以使输出更加美观。

此外，我们也可以使用 while 循环或 do-while 循环来打印九九乘法表，这里就不再赘述了，小伙伴们可以自行尝试一下。
