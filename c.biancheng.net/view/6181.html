PHP is_array()：判断是否为数组

PHP is_array() 函数用来判断一个变量是否是数组，其语法格式如下：

bool is_array ( mixed $var )

如果参数 var 是数组，就返回 true，否则返回 false。

该函数的使用示例如下： 
<?php
$a = array("C语言中文网", "PHP教程", "http://c.biancheng.net/php/");
$b = 123;
var_dump(is_array($a));
var_dump(is_array($b));
?>
以上程序的执行结果为：

bool(true)
bool(false)
