MySQL约束、函数和运算符

 
约束是一种限制，它通过限制表中的数据，来确保数据的完整性和唯一性。使用约束来限定表中的数据是很有必要的。

MySQL 提供了很多功能强大、使用方便的运算符和函数。我们可以通过使用这些运算符完成各种各样的运算操作。函数可以帮助开发人员简单、快速的编写 SQL 语句。
