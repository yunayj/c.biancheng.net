CSS教程（精讲版）

 

CSS 是「Cascading Style Sheet」的缩写，中文译为「层叠样式表」，用来控制 HTML 文档（网页）的显示样式，也即美化网页。CSS 是一种描述性的样式语言，而不是编程语言，所以没有逻辑处理能力。

这套 CSS 教程涵盖了最新的 CSS3 版本，既有入门知识，也有进阶技巧，并且附带了大量示例，将带您全面地学习 CSS。

为什么要学习 CSS？

CSS 在 Web 开发领域有着举足轻重的地位，是成为一名前端工程师必备的技能之一，下面列举了一些学习 CSS 的优势：

  • 简单易学：CSS 很容易学习和理解，并且对 HTML 文档有着强大的控制能力，所以我们通常使用 CSS 来设置 HTML 文档的样式；
  • 节省开发时间：您可以将编写好的 CSS 样式保存到一个 .css 格式的文件中，然后在多个 HTML 页面中引用；
  • 易于维护：将 CSS 样式保存在一个 .css 格式的文件中还有一个好处，就是当对网页进行改版时，您只需要修改这个 .css 格式的文件，所有应用该文件的网页都会自动更新；
  • 多种设备兼容：CSS 允许针对不同类型的设备（例如台式机，手机等）来设计网页的样式。

读者

本套 CSS 教程适合对 HTML 有一定了解的人员阅读，通过学习本教程您将拥有独立开发网页的能力。

阅读条件

在学习本教程之前，您需要具备以下能力：

  • 使用任何文本编辑器处理基本的文字；
  • 创建目录和文件；
  • 浏览不同的目录；
  • 使用当下流行的浏览器（例如 Chrome 或 Firefox）浏览网页；
  • 使用 HTML 或 XHTML 开发简单的网页。

如果您不熟悉 HTML 和 XHTML，那么建议您首先阅读我们的《HTML教程》。
