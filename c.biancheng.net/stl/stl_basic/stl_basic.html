C++ STL标准库基础

 

本章是正式进入 STL标准库学习的一道「开胃小菜」，并没有涉及具体的语法，目的是让读者对 STL标准库有一个整体上的认识，例如STL标准库是什么、STL 的作用和功能等。

STL标准库可以理解为是诸多类模板和函数模板的集合，因此读者在学习 STL标准库之前，必须对 C++ 模板有一定的了解。

大家在阅读本章教程的时候请放松心情，不用死记硬背，理解即可。
