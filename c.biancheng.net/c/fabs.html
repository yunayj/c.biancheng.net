C语言fabs()函数：求双精度浮点数的绝对值

C语言 fabs() 函数用于求双精度浮点数的绝对值。

头文件：math.h

语法/原型：

double fabs(double x);

参数 x 是求绝对值的数。

返回值：参数 x 的绝对值。

【实例】使用C语言 fabs() 函数求3和-4的绝对值。 
#include<stdio.h>
#include<math.h>
int main()
{
    double a=3.3, b=-4.4;  //为变量赋初值
    double c =fabs(a);  //求a的绝对值
    double d = fabs(b);  //求b的绝对值
    printf("%lf\t%lf\n", c, d);

    return 0;
}
运行结果：
3.300000        4.400000
