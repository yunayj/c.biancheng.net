C语言exp()函数：返回以e的x次幂（次方）

C语言 exp() 是一个指数函数，用来求 e（底数）的 x 次幂（次方）的值。

头文件：math.h

语法/原型：

double exp(double x);

参数 x 表示次幂（次方）。

返回值：e 的 x 次幂。

【实例】使用C语言 exp() 函数求双精度数 m 的指数函数值。 
#include<stdio.h>
#include<math.h>
int main()
{
    double m = 4;  //为变量赋初值
    double n = exp(m);  //求双精度数m的指数函数值
    printf("%lf\n", n);

    return 0;
}
运行结果：
54.598150
