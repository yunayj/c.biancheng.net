Redis HVALS命令

 
Redis HVALS 命令返回哈希表 key 中所有字段的值。

可用版本

HEXISTS 命令可用版本：>= 2.0.0。

语法

Redis HEXISTS 命令的基本语法如下： 
HEXISTS key field

返回值

一个包含哈希表中所有的字段值。当 key 不存在时，返回一个空列表。

命令演示

127.0.0.1:6379> HSET website name www.biancheng.net
(integer) 1
127.0.0.1:6379> HSET website name1 www.baidu.com
(integer) 1
127.0.0.1:6379> HVALS website  #返回所有的值
1) "www.baidu.com"
2) "www.baidu.com"
127.0.0.1:6379> HVALS web  #不存在返回空列表
(empty list or set)
