Redis ZREVRANK命令

 
Redis ZREVRANK 命令返回有序集合 key 中 member 的排名。其中有序集合的成员按 score 值递减(从大到小)排序，该命令排序顺序与 ZRANK 命令正好相反。

注意：排名从 0 开始， 也就是说，score 值最大的成员排名为 0 。

可用版本

ZREVRANK  命令可用版本：>= 2.0.0

语法

Redis ZREVRANK  命令的基本语法如下： 
ZREVRANK key member 

返回值

如果 member 是有序集 key 的成员，返回 member 的排名；如果 member 不是有序集 key 的成员，返回 nil 。

命令演示

127.0.0.1:6379> ZADD salary 3000 jack 3500 helen 2880 4000 simith 6000 ros
(integer) 3
127.0.0.1:6379> ZREVRANK salary jack
(integer) 3
127.0.0.1:6379> ZRANGE salary 0 -1 WITHSCORES
1) "john"
2) "2880"
3) "jack"
4) "3000"
5) "helen"
6) "3500"
7) "simith"
8) "4000"
9) "ros"
10) "6000"
