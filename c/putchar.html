C语言putchar()：输出一个字符

putchar() 函数是 C 语言标准库中的函数，定义在<stdio.h>头文件中。

putchar() 函数用于输出一个字符到标准输出设备（通常是屏幕），函数的原型如下：

int putchar(int c);

参数

函数接收一个整型参数 c，表示要输出的字符。虽然参数 c 是整型，但通常只使用其最低的 8 位来表示 ASCII 字符。

返回值

如果函数执行成功，则返回要输出的字符 c；如果发生错误，则返回 EOF。

【实例】用 putchar() 函数打印一串字符，请看下面的 C语言代码。
#include <stdio.h>

int main() {
  const char *str = "http://c.biancheng.net";
 
  // 使用循环和 putchar() 打印字符串中的每个字符
  for (int i = 0; str[i] != '\0'; i++) {
    putchar(str[i]);
  }

  // 打印换行符
  putchar('\n');

  return 0;
}
输出结果为：

http://c.biancheng.net

该代码使用 putchar() 函数将字符串 "http://c.biancheng.net" 的每个字符打印到标准输出，然后打印一个换行符。
