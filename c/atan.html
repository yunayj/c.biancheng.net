C语言atan()：求反正切

atan() 是 C语言的一个标准库函数，定义在<math.h>头文件中。

atan() 函数用于计算给定数字 x 的反正切值（反三角函数），函数的原型如下：
double atan(double x);

参数

x：计算反正切值的浮点数。

返回值

返回 x 的反正切值，以弧度为单位。

【实例】以下的 C 语言示例代码展示了用 atan() 函数计算一个数值的反正切值，并打印结果。
#include <stdio.h>
#include <math.h>

int main() {
  double value = 1.0;
  double result;

  // 计算1.0的反正切值
  result = atan(value);

  printf("atan(%f) = %f 弧度\n", value, result);

  return 0;
}
输出结果为：

atan(1.0) = 0.785398 弧度
