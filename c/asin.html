C语言asin()：求反正弦

asin() 是 C语言的一个标准库函数，定义在<math.h>头文件中。

asin()函数用于计算给定数字 x 的反正弦值（反三角函数），函数的原型如下：
double asin(double x);

参数

x：范围在 [−1，1] 的浮点数。

返回值

返回 x 的反正弦值，以弧度为单位。

【实例】以下的 C 语言示例代码展示了用 asin() 函数计算一个数值的反正弦值，并打印结果。
#include <stdio.h>
#include <math.h>

int main() {
  double value = 0.5;
  double result;

  // 计算0.5的反正弦值
  result = asin(value);

  printf("asin(%f) = %f 弧度\n", value, result);

  return 0;
}
输出结果为：

asin(0.5) = 0.523599 弧度
