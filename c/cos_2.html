C语言cos()函数：求余弦

C语言 cos() 函数用来求余弦值。

头文件：math.h

语法/原型：

double cos(double x);

参数 x 是一个弧度值。

返回值：余弦函数值。

【实例】使用C语言 cos() 函数求 m 的余弦函数值。 
#include<stdio.h>
#include<math.h>
int main()
{
    double pi = 3.1415926;  //为变量赋初值
    double m = pi / 4, n;  //为变量赋初值
    n = cos(m);  //求m的余弦函数值
    printf("%lf\n", n);

    return 0;
}
运行结果：
0.707107
