某大厂码农说：拓展社交圈子只需要4个字

程序员是一个比较宅的群体，大部分时间都和电脑苟在一起，很多人对社交圈子比较渴望，但是又不知道如何切入。

大部分人的感觉就是，年龄越大，圈子越重要。我们的生活中有很多琐事不能由自己控制，只能借助圈子打破困境。如何培养自己的圈子，是每个社会人一辈子都逃不过的话题。

首先我们要知道，圈子的本质是相互帮助，而不是某个人的单向输出，要想扩大自己的圈子，核心要领就是提高自己的“被利用”价值，让别人从你身上“沾光”。那些一味利用别人，或者请求帮忙的人，要么是弱者，要么是 loser。

被利用价值有很多方面：
  • 你可以是一个有趣的灵魂，满身都是幽默细胞，总能在恰当的时机活跃气氛。
  • 你可以是一个有钱人，能在他人落魄的时候提供经济支持。
  • 你可以是一个枢纽，能够连接更多人，帮助别人牵线搭桥。
  • 你可以拥有一部豪车，载着他人出席重要场合。
  • 你可以拥有一艺之长，帮助他人做点小事，省点小钱。
  • 你可以拥有调配资源的能力，为靠谱的公司/个人带来经济收入。

明白了这个底层逻辑，我们就有了进步的方向，那么接下来，我们所要做的就是不断提高自己的被利用价值。

这是一个比较长期的话题，不是立马就能解决的，你需要好好规划自己的人生路线，积攒足够的资源、人脉、技能或者金钱，然后才能拥有超出常人的价值。

但是当下的现实却是另一个版本：大部分人都是芸芸众生，除了掌握点小技能，认识个芝麻官，就剩下一腔热情了，你可能十年或者一辈子都没有一个值得称赞的成就。

就算你拥有亮眼的价值，别人都羡慕你，一味地坐等往往也不容易扩大社交圈子。

我曾经一度被这个问题困惑，不知道如何在老家积累自己的人脉。后来一个 90 后程序员（人称泥瓦匠，也就是搬砖的）从杭州过来找我面交，我知道他认识很多创业者、程序员、设计师等各种角色，所以在他即将返程，我送他去车站的路上，向他抛出了这个问题。

由于快要发车了，他也没有机会说太多，最后只送了我四个字：

主动+利他

这四个字说得真是太在理了！虽然看起来像鸡汤，但是落地后却像核武器。

大部分人社交首先想到的是别人对自己有什么价值，能帮助自己解决什么困难，以及如何才能从别人身上捞取好处。这容易让人保持距离，不敢深交。

但是，如果我们换位思考一下，在恰当的时机主动向别人伸出援助之手，或者在一个圈子里面主动做点力所能及的小事，那是不是更容易受到别人的喜欢和信任。你帮助了别人，或者付出了劳动，别人是不是也惦记着给你方便？至少当你向别人求助时，别人不会轻易拒绝你吧。

世界上最难偿还的不是债务，而是情分。有些情分可能需要挂念一辈子，用一生的时间来偿还。别人在帮助你时并不一定期望你的回报，但是你一定会有一颗感恩的心，时不时地惦记这件事，一旦有机会你就会报恩。即使不能报恩，你也会逢年过节去探望，或者到处传播这个人的美名。

与人方便，手有余香，说的就是这个道理。

我一直不理解为什么有些老板喜欢打肿脸充胖子，必须要买一部豪车，后来我才明白，他们并不一定是想撑场面，或者想彰显地位，很有可能他们只是想帮助别人，因为有很多场合都需要一部豪车。

作为一个老板，你能提供的最大价值就是物质，除了借钱，你还可以借车。

聊聊程序员

作为一个程序员，我们资源有限，可能无法给与别人那种改变人生发展轨迹的帮助，但是有很多力所能及的小事，我们还是可以指点一二的，比如：
  • 有人不知道 Java 如何学习，你可以吧啦吧啦给他讲半天。
  • 有人想买电脑或手机，你可以给点建议。
  • 有人遇到 bug 了，你可以一起调试。
  • 有人想入坑编程，你可以说一下这个行业的利弊。
  • 有人问你技术问题，你可以尽量回复。
  • 有人路由器不能用了，你可以帮忙设置一下。
  • 有人想创业，你可以帮忙分一下技术路线。

凡此种种，都是可以让你积累资源的。它并不一定立马就奏效，但是在未来的某一天，或者等到小白成长起来，你就会得到反哺。

如果你想加入一个更高的圈子，可能就无法凸显自己的价值了，这个时候你可以做点力所能及的小事：
  • 比如在一个创业者的私人聚会上，你可以帮忙做饭，或者帮忙买菜，然后顺便咨询一下别人的口味，让别人记住你。
  • 比如老家有菜园子，你可以采摘一些绿色有机蔬菜，路过领导小区的时候，顺便给领导带过去。

就算你自己提供不了价值，你还有家人和朋友，他们也可以帮你提供额外的价值，比如：
  • 领导的孩子正在择校，而你恰好有家人是老师，你就可以让家人提供一些建议。
  • 同事牙疼，而你恰好有朋友是牙医，你也可以请教一下你的朋友。

不要总是担心麻烦别人，我们就是在相互麻烦之时促进感情的。当然，请求别人帮忙一定要带着诚意和热情。

我和泥瓦匠

其实，我和泥瓦匠的感情基本也是这么建立的，他先向我伸出了援助之手，然后我又给他分享了一些小道消息，一来二去，我俩就成了可以互开玩笑的好朋友。

泥瓦匠有着非常系统的产品思维，他从杭州来到我的老家，竟然帮我规划起了C语言中文网的产品定位以及用户体验，而且在高铁上写了一路的思维导图。

当时我整个人是非常震惊的，这特么是要做什么？虽然我知道他可能也想问我一点事情，但是这个成本也太高了吧。在不解之余，我当然也是满怀感激。

后来我知道他也想运营一个网站，做一个自由职业者，我就决定主动联系他，给他分享一些 SEO 干货。

从大学到现在，我已经从事 SEO 十多年了，踩了很多坑，交了很多学费，最终才理解了 SEO 的底层逻辑，掌握了一些流氓打法。这些“绝活”我一直都憋在心里，舍不得跟人交流，结果那天头脑一热，全都输出给泥瓦匠了。

其实我并不是头脑发热，我只是觉得必须要这么做。

经过这次赤诚的交流，我俩很快就建立了信任关系，变成了可以一起做事的好朋友。

再后来，我运营了公众号，粉丝积累还不错，我俩就计划着一起做点事情。
